# Setup

1. Install the dependencies
```
yarn
```

2. Create the .env file
```

POLYGON_MATIC_VIRGIL=XXXXXXXXXXXXXXXX
ETHERSCAN_API_KEY=XXXXXXXXXXXXXXXX
POLYGONSCAN_API_KEY=XXXXXXXXXXXXXXXX
```

3. Run your farm
```
yarn run build && yarn run start
```
